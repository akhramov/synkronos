# Synkronos

    Synkronos is a life-time folder synchronization tool. 
    It's based on rsync and works only under MacOSX yet.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'synkronos'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install synkronos
